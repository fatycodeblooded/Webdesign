Languages for creating websites

HTML

HTML stands for Hyper Text Markup Language. HTML is the standard markup language for creating Web pages. HTML describes the structure of a Web page.

CSS

CSS is the language for describing the presentation of Web pages. CSS is independent of HTML and can be used with any XML-based markup language.

JavaScript

JavaScript is a text-based programming language used both on the client-side and server-side that allows you to make web pages interactive.

C++

C++ can be used to develop operating systems, browsers, games, and so on.
